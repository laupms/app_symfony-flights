Projet "flights"
Application de gestion de vols aériens
- mise en place de rôles utilisateur avec formulaire de login/connexion
- méthode CRUD pour la gestion des vols, des aéroports, des comapgnies aériennes, des villes, des réductions et des utilisateurs
- formulaires sécurisés d'ajout et de modification des éléments (contraintes)
- données stockées en base de données relationnelles MySQL
- responsive design : Bootstrap, menu burger, media queries, JS
- routes sécurisées selon l'utilisateur
- fixture pour les tags des vols (relation ManyToMany avec Flights)
- pages d'erreur
- pages légales
- référencement SEO (mots-clés, balises meta...)
